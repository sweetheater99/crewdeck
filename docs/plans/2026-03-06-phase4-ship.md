## Task 1: Docker One-Click Deploy

Overhaul the existing Dockerfile for a clean single-container experience.

**Modify:** `Dockerfile`
- Multi-stage build: install deps → build → production image
- Bundle: Node.js + built server + built UI + embedded PostgreSQL
- Single command: `docker run -p 3100:3100 -v crewdeck-data:/root/.crewdeck crewdeck`
- Health check endpoint
- Sensible defaults (no env vars required for basic usage)

**Modify:** `docker-compose.yml`
- Simple compose for local dev with external Postgres
- Comment out embedded mode option

**Create:** `docker-compose.quickstart.yml`
- Overwrite existing with a clean quickstart:
```yaml
services:
  crewdeck:
    image: crewdeck/crewdeck:latest
    ports: ["3100:3100"]
    volumes: ["crewdeck-data:/root/.crewdeck"]
    restart: unless-stopped
volumes:
  crewdeck-data:
```

**Update:** README.md quickstart section with Docker instructions.
